# Breakout [OpenGL]
Breakout clone written in C using OpenGL. The waves move while you're playing. :-)

![Game running](img/Breakout.png)

## How to run?

If you want to run the code, install the following packages:

```
install-package nupengl.core
install-package glm
```

## Commands

- [P] for playing
- [A] move left
- [D] move right
- [R] for restart
